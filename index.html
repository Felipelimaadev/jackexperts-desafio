Apresentação do Projeto

Este projeto faz parte de um desafio para criar uma aplicação simples com uma página HTML customizável, definida via Helm e hospedada em um cluster Kubernetes. A aplicação utiliza Docker para empacotar o ambiente e Helm para gerenciar a implantação no Kubernetes.

Repositórios

Você pode acessar os repositórios do projeto nos links abaixo:

  • GitHub - Código Fonte
  • Docker Hub - Imagem Docker

Tecnologias Utilizadas

As seguintes ferramentas e tecnologias foram usadas para completar este desafio:

  • Docker
  • Kubernetes
  • Helm
  • Git
  • Docker Hub

Desafios e Facilidade no Projeto

O que foi mais difícil durante a implementação do projeto:

  • Configurar o Helm corretamente para criar todos os objetos do Kubernetes de forma automatizada.
  • Garantir que o contêiner não rodasse como root, ajustando as permissões corretamente no Dockerfile.

O que foi mais fácil durante a implementação:

  • A construção da imagem Docker e a publicação no Docker Hub.
  • A customização da página HTML para refletir as informações do projeto.
